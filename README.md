# alcock.net

A home for the Alcock name — plus jason.alcock.net, the personal site of its
first resident.

- `site/` — **alcock.net**: home, famous Alcocks, history of the name, join
  page (mailto-based; Jason listed as founding member). Parchment theme,
  rooster motif after Bishop John Alcock's rebus.
- `jason/` — **jason.alcock.net**: dark cousin of the same theme. OurQA,
  Breakbone Addons, punchjohneric, Catalyst, corp-engine, contact.
- Static HTML/CSS only. No build step, no JS, no trackers.

Deploy: see `DEPLOY.md` / `deploy.sh` (same conventions as ourqa-site — no
stored credentials, rsync with interactive password).
